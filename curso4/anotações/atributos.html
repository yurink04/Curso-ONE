O que são atributos em tags?

Os atributos são comandos ou informações específicas que damos a uma tag para que ela faça tal comando, porém, com algum detalhe ou variação específica em realação à tag original, sem atributos.

Para que são utilizados?

Eles são utilizados para fornecer informações adicionais sobre um elemento HTML.

Como são aplicados?

Os atributos HTML são inseridos dentro da Tag especificada, fornecendo instruções ao navegador sobre como exibire interagir com o elemento.

Qual a necessidade de uso?

Os atributos personalizam as tags em HTML, modificando sua estrutra ou funcionalidade.
Por exemplo, os atributos podem:

  1. Atribuir uma classe ou ID a um elemento, para facilitar a estilização com CSS ou a maniupulação com JavaScript.
  2. Definir o tamanho, a cor, o alinhamento, o título, o idioma ou o texto alternativo de um elemento.
  3. Indicar o tipo, o valor, o nome, o destino ou a ação de um elemento de formulário.
  4. Especificar a fonte, o destino, o método ou o relacionamento de um elemento de link.
  5. Informar o caminho, o formato, a largura, a altura ou o controle de um elemento de mídia.

Alguns atributos são genéricos e podem ser usados na maioria das tags, enquanto outros são específicos para certos elementos.

Atributo Class: Para que serve?

O atributo class em HTML serve para especificar um ou mais nomes de classe para um elemento HTML. Os nomes de classe podem ser usados por CSS e JavaScript para selecionar e acessar elementos específicos através de seletores de classe ou funções como o método DOM document.getElementByClassName.
O atributo pode ser usado class pode ser usado em qualquer elemento HTML, permitindo estilizar elementos com propriedades comuns.
Por exemplo, se você quiser aplicar uma cor e um tamanho de fonte maior para todos os elementos que tenham a classe "destaque", é possível utilizar CSS.
