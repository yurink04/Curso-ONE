Reset CSS: O que é?

O Reset.CSS ou o CSS Reset, é uma técnica ou conjuntos de estilos CSS projetados para normalizaro estilo padrão dos elementos HTML em todos os navegadores da web.
O objetivo é criar uma base consistente e previsível para o desenvolvimento web, independentemente das variações de estilo padrão entre diferentes navegadores.

O Reset CSS ajusta o layout, resetando tags que possuíam valores iniciais (que ocasionavam uma pré-formatação), nos retornando um estilo básico em tags como h1, a, body e etc. Esta técnica suaviza as diferenças e padroniza a estilização, sobrepondo a formatação original dos browsers com uma folha de esilo.

Como criar o Reset CSS?

Para reduzirmos as formatações originais entre os navegadores, é preciso selecionar todas as tags e adicionar sobre elas os resets, como margin, padding, border e etc.

A formatação deve ser criada de acordo com a necessidade do seu projeto, ou pode ser utilizado algum reset já pronto.
Um dos mais utilizados e conhecidos foi desenvolvido por Eric Meyer. No seu blog, onde ele explica um pouco sobre a técnica e como desenvolveu o seu código reset.

Note que existem algumas tags que são dividas em grupos, que recebem valores genéricos, como margin: 0, border: 0 padding: 0 e outras que precisam de valores específicos, como:

ol, ul { list-style: none; }
