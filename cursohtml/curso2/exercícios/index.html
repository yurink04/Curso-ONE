Lista de exercícios: Mão na Massa

Esta lista de exercícios é um caminho prático e envolvente de aprender e aprimorar as suas habilidades em tecnologia.

Como fazer?

  1. Leia com atenção cada exercício proposto: comece entendendo o cenário projetado e o que se espera como resultado
  2. Escreva o código: utilizade o ambiente de códificação para digitar e programar o seu código.
  3. Verifique o gabarito: após tentar resolver o exercício, comprare o seu código com o gabarito fornecido em "Opnião do instrutor.
  4. Repita: a repetição é a chave para o aprendizado.

1) Estilizando os elmentos com clases CSS

Você está trabalhando no desenvolvimento de uma página web e precisa aplicar estilos específicos a diferentes elementos usando classes no CSS. Seu desafio é criar uma classe chamada "texto destaque" no arquivo HTML para um elemento p e, em seguida, estilisar essa classe no arquivo CSS para alterar a cor do texto.

Destacanto Títulos com CSS

Imagine que você está desenvolvendo um blog e precisa destacar os títulos das postagens. Crie uma classe CSS chamada "titulo-blog" e aplique-a a elementos de h1 no seu arquivo HTML. Em seguida, personalize essa classe no arquivo CSS para mudar a cor do texto, para dar mais destaque.
