Emmet Abbreviation - Atalhos de produtividade

Inserindo atributos ou elementos nas tags

Para inserir um atributo ou elemento durante a criação de uma TAG com abreviação Emmet, é necessário usar >. Por exemplo:
Ao criar uma lista, para declarar os iten dela:
*ol>li*
Ao criar uma DIV, para inserir um h1 ou p:
*div>h1*
Ao criar uma lista com href:
*ul>(li>a{conteúdo$})*

Inserindo conteúdo nas tags

Para inserir conteúdo nos atribus das tags, usa-se {}. Por exemplo:
Ao criar uma lista com itens, para declara seu contéudo:
*ol>li{Aqui eu posso colocar conteúdo}*
Ao criar uma div, para declarar h1 ou p:
*div>h1{Aqui eu posso colocar conteúdo}*
Ao criar uma lista com link, para declarar seu conteúdo:
*ul>(li>a{conteúdo$}[href:"http:link.com/conteúdo-$"])*
